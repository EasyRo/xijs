## arrayToListNode

> 数组转成链表

Demo:

```tsx | pure
import {arrayToListNode} from 'xijs'

const arr = [1, 2, 3]
console.log(arrayToListNode(arr))
const arr1 = []
console.log(arrayToListNode(arr1))

```
